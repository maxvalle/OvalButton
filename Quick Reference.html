A canvas based control with many options to mimic the Apple Safari buttons used for bookmarks bar.


this class make use of HierPop classes which are

Copyright by Noah Desch <http://www.wireframesoftware.com/>



Properties


Caption as string

The button caption


Menu as HCMenu

The associated menu. (May be NIL)


Selected as boolean

Set/unset the value for Sticky and Toggle Buttons



Properties (only availables in UI design)


Align as integer

The button alignment (0=left, 1=center, 2=right)


CheckedMenu as boolean

Flag to indicate if the menu should use checkmarks for selection.


FixedSize as boolean

Flag to indicate the button should not auto resize depending on caption length


MenuOnRight as boolean

Flag to indicate if the menu should be displayed on right of control.


Type as integer

The button type (0=PushButton, 1=StickyButton, 2=ToggleButton, 3=MenuButton)




Methods


none



Events


Action(menuItem as ConItem)

This event is fired when the button is pressed or, in case of MenuButton, a menu item is selected. The menuItem parameter contains the selected menu Item or NIL in case of button without menu.


To handle an associated menu, see the example or the HierPop documentation.
